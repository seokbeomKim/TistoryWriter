티스토리 블로그에 글을 포스팅할 수 있도록 도와주는 플러그인입니다.

== Description ==

사용자의 티스토리 계정을 연동하여 새 글을 작성 시 발행 단계에서 포스팅할 티스토리 블로그의 카테고리를 선택할 수 있습니다.

기존 글들을 불러오는 기능은 아직 구현되지 않았습니다.

== Installation ==

다음은 플러그인 설치법입니다.

1. `/wp-content/plugins/` 디렉토리에 TistoryWriter 압축을 해제합니다.
2. 워드프레스의 Plugin 메뉴를 선택하여 플러그인을 활성화합니다.
3. 추가한 Tistory Writer를 클릭하고 티스토리 계정을 연동합니다.
4. 연동이 완료되면 글 작성 시마다 등록할 블로그의 카테고리 선택버튼이 활성화됩니다.
