# Tistory Writer (개발중)
워드프레스용 티스토리 연동 플러그인 입니다.
플러그인은 개발 중이며 사용 중 발견되는 버그는 <sukbeom.kim@gmail.com>으로 메일 주시기 바랍니다.

## 연동 기능

### 가. 포스팅 위치 선택
포스팅하고자 하는 티스토리의 블로그, 카테고리를 선택할 수 있습니다.
* 블로그 선택
* 포스팅용 카테고리 선택

### 나. 글 옵션 선택
* 공개 또는 비공개
* 태그 및 위치 설정
* CCL (저작물 사용 허가 표시)
* 댓글 허용 권한
* 주제 선택 기능
